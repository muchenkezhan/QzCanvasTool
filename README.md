# QzAiTool

QzAiTool 是一个面向 AI 创作、图像处理和可视化工作流的桌面软件。它把 AI 对话、提示词管理、图像生成、图像编辑、高清放大、智能抠图、插件工具和节点式工作流整合在同一个工作台里，适合电商设计、批量素材生产、创意辅助和本地效率工具场景。

## 软件定位

QzAiTool 的核心不是单一 AI 聊天窗口，而是一个可组合的 AI 创作工作台。用户可以把文本、图片、视频、提示词、模型调用和处理工具拆成节点，通过连线形成可复用流程，再按需要单次运行、批量运行或保存为模板。

## 核心能力

- **节点式工作流**：通过画布节点组织输入、处理、生成和输出步骤，支持拖拽、连线、编组、框选、复制粘贴、模板保存和工作流导入导出。
- **画布 AI 助手**：通过自然语言创建、调整和运行工作流，可读取当前画布选区，把选中的图片节点作为参考图参与生图、改图或流程编排。
- **多模型创作**：区分聊天、图片、视频等模型配置，支持不同供应商和协议能力接入。
- **图像生成与编辑**：支持文生图、参考图生图、局部修改、扩图、裁切、属性修改、批量提示词生图等常用图像创作流程。
- **图像处理节点**：包含高清放大、透明抠图、宫格裁切、图像拼接、图层与文字拆分、位图转矢量等处理能力。
- **媒体库**：集中管理生成或导入的图像、视频资源，支持收藏、定位、复制资源、批量操作和放入工作流。
- **提示词广场**：沉淀常用提示词和预设方案，降低重复输入成本。
- **应用中心**：支持本地 app 插件，扩展图片压缩、水印生成、配色、变量命名等独立工具。
- **内置浏览器**：提供独立浏览器视图，方便在软件内打开辅助页面或参考资料。

## 工作流画布

工作流画布是 QzAiTool 的主要生产界面。画布中的节点以卡片形式存在，节点之间通过关系线传递文本、图片、视频或执行依赖。

常用操作包括：

- 添加文本、图片、模型、处理、预览和保存类节点。
- 通过连线把上游输出作为下游输入。
- 框选多个节点后进行对齐、编组、批量运行或复制粘贴。
- 将常用节点组合保存为工作流模板。
- 把媒体库资源、上传文件或粘贴图片直接放入画布。
- 使用画布助手通过对话生成或调整节点流程。

## 自定义供应商接入教程

QzAiTool 支持把对话、图片、视频供应商分开配置。不同类型的模型不要混在同一个配置里：聊天模型放到“对话模型”，生图/改图模型放到“图片模型”，视频生成模型放到“视频模型”。

### 推荐供应商

如果需要使用中转或聚合类模型服务，可以优先查看：

- [https://api.aistat.top/](https://api.aistat.top/)
- [https://apitrans.top/](https://apitrans.top/)

接入前先在供应商后台确认 API Key、接口地址、可用模型 ID、计费规则和支持的协议类型。

### 添加对话供应商

1. 打开系统设置，进入“供应商配置”。
2. 切到“对话模型”，点击“添加对话供应商”。
3. 选择“添加自定义供应商”。
4. 填写供应商名称、API 地址、协议、API Key。
5. 在“模型”里每行填写一个模型 ID，例如供应商后台显示的聊天模型 ID。
6. 添加后可以在供应商详情里点击“测试连接”或“拉取模型”。

常用协议选择：

- OpenAI 兼容聊天接口：选择 `OpenAI Chat`。
- OpenAI Responses 接口：选择 `OpenAI Responses`。
- Claude 原生 Messages 接口：选择 `Anthropic Messages`。
- Gemini 原生接口：选择 `Gemini Generate Content`。
- 同一个聚合网关里混合多种端点：选择 `聚合网关`，再给每个模型指定端点类型。

### 添加图片供应商

1. 打开系统设置，进入“供应商配置”。
2. 切到“图片模型”，点击“添加图片供应商”。
3. 选择“添加自定义供应商”。
4. 填写供应商名称、API 地址、协议、API Key。
5. 在“模型”里每行填写一个图片模型 ID。
6. 添加后进入供应商详情，为图片模型补充比例、分辨率和高级选项。

图片协议选择建议：

- OpenAI Images 兼容生图接口：选择 `OpenAI Images`。
- AI SDK Image Generation 接口：选择 `Image Generation`。
- OpenAI Responses 生图接口：选择 `OpenAI Responses 生图 (/v1/responses)`。
- Gemini 原生生图/改图接口：选择 `Gemini Generate Content`。
- 使用聊天接口返回图片：选择 `OpenAI Chat Image`。
- Agnes 图片接口：选择 `Agnes Images`。

图片模型里的“模型映射”用于解决供应商展示模型 ID 和真实能力模型 ID 不一致的问题。普通情况下留空即可；只有当供应商要求传入另一个真实模型 ID 时再填写。

### 添加视频供应商

1. 打开系统设置，进入“供应商配置”。
2. 切到“视频模型”，点击“添加视频供应商”。
3. 选择“添加自定义供应商”。
4. 填写供应商名称、API 地址、协议、API Key。
5. 在“模型”里每行填写一个视频模型 ID。
6. 如果供应商需要额外参数，在“高级选项”里填写 JSON 参数。

视频协议选择建议：

- 统一视频网关：选择 `AI SDK Gateway（统一视频）`。
- 通用异步任务式接口：选择 `通用任务式视频`。
- OpenAI 视频直连：选择 `OpenAI Video 直连 (/v1/videos)`。
- OpenAI 兼容视频生成：选择 `OpenAI 兼容视频生成 (/v1/videos/generations)`。
- Grok/xAI 视频生成：选择 `Grok/xAI 视频生成 (/v1/video/create)`。
- 原生视频平台：按供应商选择 `Gemini Veo`、`Kling 原生视频`、`Runway 原生视频`、`Minimax/Hailuo 原生视频`、`Luma Dream Machine 原生视频` 或 `Agnes Video`。

### API 地址填写规则

- 优先填写供应商后台给出的 Base URL。
- OpenAI 兼容协议可以填写根地址或带 `/v1` 的地址，界面会显示最终请求预览。
- 不确定是否填写正确时，先看“预览”行是否拼出了预期接口，例如 `chat/completions`、`responses`、`images/generations` 或视频生成路径。
- 不要把聊天、图片、视频接口强行复用成同一个供应商配置；按模型能力分别添加，后续排查问题更清楚。
- API Key 属于敏感信息，只在软件配置页填写，不要写入文档、截图或工作流模板。
